## 📖 Course Description

This repository contains **in-class exercises** developed during the lectures of the **Programming 3** course offered at PUCP during the 2026-1 academic semester.

The course is of a theoretical-practical nature, whose purpose is for the student to understand the implementation and application of enterprise programming concepts. Topics developed include foundations of Java & C#, advanced object-oriented programming, software modularity, database persistence, core web technologies, GUI development, distributed systems, data reporting, and multithreading.

## ⚙️ Getting Started

### Prerequisites

Ensure you have the following tools installed:

- **Git** ≥ 2.30
- **Java Development Kit (JDK)** 25
- **C# / .NET SDK** 10.0
- **IDE** (IntelliJ IDEA and Visual Studio are recommended since they are installed in the labs)

### Clone the Repository

```bash
git clone https://github.com/jlcorcuera/pucp-progra3-2026-01.git
cd pucp-progra3-2026-01
```
